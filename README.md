# Анализ падения конверсии в МобиКар

## Задача
Проанализировать падение конверсии пользователей с первого входа в приложение до первой поездки на 15% за 2 месяца.

## Подход
1. Проверил влияние будних/выходных дней — разница <5%
2. Разбил данные по неделям, построил воронку:
   - Установка - Регистрация - Поиск - Бронирование - Поездка
3. Нашёл аномалию: на 7-й неделе конверсия **Регистрация - Поиск** упала с 70.8% до 49.3%

## Гипотезы
- Обновление поиска перед 7-й неделей
- Праздничные бонусы (23 февраля) - нецелевые регистрации
- Возможная потеря данных

## Дальнейшие шаги
- A/B тест старого vs нового поиска
- Когортный анализ регистраций по бонусам
- Проверка технических логов (ошибки при поиске)

## Стэк

- matplotlib

## Результат
Выявлен шаг воронки и неделя, где произошло падение. Предложены проверяемые гипотезы.
